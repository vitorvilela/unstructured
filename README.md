# unstructured

- Triangle: Engineering a 2D Quality Mesh Generator and Delaunay Triangulator
http://www.cs.cmu.edu/~quake/triangle.research.html
- Uthash: A hash table for C structures
https://troydhanson.github.io/uthash/
